**lib/ws/server.js** -- Custom HTTP server that wraps Next.js:
- Import createServer from 'http', parse from 'url', next from 'next', WebSocketServer from 'ws'
- Import validateTicket from './tickets.js' and proxyToTtyd from './proxy.js'
- Create Next.js app (`next({ dev: process.env.NODE_ENV !== 'production' })`), await app.prepare()
- Create HTTP server, delegate normal requests to Next.js handle()
- Create WebSocketServer in noServer mode
- On server 'upgrade' event:
  1. Parse URL, only handle paths starting with `/ws/terminal/`
  2. Extract `ticket` query parameter
  3. Call validateTicket(ticket) -- if null, write 'HTTP/1.1 401 Unauthorized\r\n\r\n' and socket.destroy()
  4. Check Origin header against process.env.APP_URL as secondary CSWSH defense -- if mismatch, 403 and destroy
  5. Call wss.handleUpgrade(req, socket, head, cb) then emit 'connection' with ticketData
- On wss 'connection', call proxyToTtyd(ws, ticketData)
- Listen on PORT env var (default 80)

**lib/ws/proxy.js** -- Bidirectional WebSocket proxy:
- Import WebSocket from 'ws', Docker from 'dockerode', getWorkspace/updateWorkspace from '../db/workspaces.js'
- Create docker instance with socketPath /var/run/docker.sock
- Export async function proxyToTtyd(clientWs, ticketData):
  1. Get workspace from DB by ticketData.workspaceId
  2. If not found or status !== 'running', close clientWs with code 4404
  3. Resolve container IP: docker.getContainer(ws.containerId).inspect() -> NetworkSettings.Networks[`${ws.instanceName}-net`].IPAddress
  4. If no IP, close clientWs with code 4500
  5. Connect upstream WebSocket to `ws://${ip}:${ticketData.port}/ws`
  6. On upstream open, updateWorkspace lastActivityAt
  7. Bidirectional message relay with `{ binary: isBinary }` to preserve ttyd binary protocol
  8. Close propagation both directions
  9. Error handling: log and close counterpart

**lib/ws/actions.js** -- Server Action for authenticated ticket issuance:
- 'use server' directive
- Import auth from '../auth/config.js', issueTicket from './tickets.js', getWorkspace from '../db/workspaces.js'
- Export async function requestTerminalTicket(workspaceId, port = 7681):
  1. const session = await auth(); if no session.user, throw Unauthorized
  2. const ws = getWorkspace(workspaceId); if not found or not running, throw error
  3. Return { ticket: issueTicket(workspaceId, port, session.user.id) }
