## Decisions Made
- Used direct fetch() with `Accept: application/vnd.github.raw+json` instead of githubApi() which hardcodes `application/vnd.github+json` (returns base64 instead of raw text)
- Both 404 and 403 return null gracefully -- repos without .planning/ directory are expected, not errors
- Truncation limits: STATE.md at 4000 chars, ROADMAP.md at 6000 chars to prevent context bloat in the agent
