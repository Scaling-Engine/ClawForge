```bash
REPO_STATE_MD=""
if [ -f "/job/.planning/STATE.md" ]; then

    fi
fi

REPO_ROADMAP_MD=""

    if [ "${#RAW}" -gt 6000 ]; then

```

**Step 8e: Read git history (after Step 8d)**

Fetch main branch history from the shallow clone (Research Pitfall 1 -- the clone is `--single-branch --depth 1` so main is NOT available without fetching). Then extract last 10 commits.

```bash
# 8e. Read recent git history from main branch
GIT_HISTORY=""
git fetch origin main --depth=11 2>/dev/null || true
GIT_HISTORY=$(git log origin/main --oneline -n 10 --format="- %h %s (%cr)" 2>/dev/null || echo "")
```

Use `|| true` on fetch so repos where main doesn't exist (unlikely but safe) don't crash the script.
