**lib/chat/components/ui/combobox.jsx** — Cherry-pick from upstream `https://raw.githubusercontent.com/stephengpope/thepopebot/main/lib/chat/components/ui/combobox.jsx`.

This is a fully custom React component (NO Radix UI). Props:
- `options` — array of `{ value: string, label: string }`
- `value` — currently selected value
- `onChange` — callback when selection changes
- `placeholder` — input placeholder text
- `loading` — shows loading state
- `disabled` — disables interaction
- `highlight` — highlights the component border

Component behavior:
- Click to open dropdown
- Type to filter options via case-insensitive `String.includes()`
- Click option to select, fires onChange
- Click outside to close (useEffect with document click listener)
- Selected option shows CheckIcon
- Search input has SearchIcon prefix
- Dropdown toggle has ChevronDownIcon

Imports:
- `{ useState, useRef, useEffect }` from `'react'`
- `{ SearchIcon, ChevronDownIcon, CheckIcon }` from `'../icons.js'`
- `{ cn }` from `'../../utils.js'`

Do NOT add any Radix UI dependencies. Do NOT use lucide-react.

**lib/chat/components/tool-names.js** — Write fresh for ClawForge.

Export `getToolDisplayName(toolName)` function that:
1. Replaces underscores with spaces
2. Capitalizes the first letter of each word (Title Case)
3. Returns the formatted string

Example: `'create_job'` -> `'Create Job'`, `'get_system_technical_specs'` -> `'Get System Technical Specs'`

This is an auto-formatter, NOT a static map. No imports needed.
