    No npm dependencies needed. All browser-native APIs.

    1. Create `public/voice-processor.js` — AudioWorklet processor (static file, NOT bundled):
       - Class `VoiceProcessor extends AudioWorkletProcessor`
       - `process(inputs)` method:
         - Get `input = inputs[0]?.[0]`, return true if no input
         - Convert Float32 samples to Int16 PCM: `pcm16[i] = Math.max(-32768, Math.min(32767, Math.round(input[i] * 32767)))`
         - Calculate volume: `Math.max(...input.map(Math.abs))`
         - Post message: `this.port.postMessage({ pcm16, volume })`
         - Return true (keep processor alive)
       - `registerProcessor('voice-processor', VoiceProcessor)`

    2. Create `lib/voice/config.js` — Feature detection (VOICE-03):
       - `export function isVoiceSupported()` — returns boolean
         - Check `typeof window !== 'undefined'`
         - Check `navigator.mediaDevices?.getUserMedia`
         - Check `typeof AudioWorkletNode !== 'undefined'`
         - Return true only if all three exist
       - This is used by chat-input to conditionally render the mic button

    3. Create `lib/voice/recorder.js` — AudioWorklet microphone capture (VOICE-01, VOICE-04):
       - `export async function startMicCapture({ onAudioData, onVolume, onError })`
       - Wrap everything in try/catch:
         - `const stream = await navigator.mediaDevices.getUserMedia({ audio: { sampleRate: 16000, channelCount: 1, echoCancellation: true } })`
         - `const audioContext = new AudioContext({ sampleRate: 16000 })`
         - `await audioContext.resume()` (handle autoplay policy)
         - `await audioContext.audioWorklet.addModule('/voice-processor.js')`
         - Create source from stream, create AudioWorkletNode 'voice-processor'
         - `processor.port.onmessage = (event) => { onAudioData(event.data.pcm16.buffer); onVolume?.(event.data.volume); }`
         - `source.connect(processor)` — do NOT connect processor to destination (no echo)
         - Return `{ stop() }` that: disconnects processor, disconnects source, stops all stream tracks, closes audioContext
       - Catch block (VOICE-03):
         - `NotAllowedError` -> `onError?.('Microphone permission denied')`
         - `NotFoundError` -> `onError?.('No microphone found')`
         - Other -> `onError?.('Failed to start microphone')`
         - Return null on error

    4. Create `lib/voice/transcription.js` — AssemblyAI v3 WebSocket client (VOICE-02):
       - `const WS_URL = 'wss://streaming.assemblyai.com/v3/ws'`
       - `export function createTranscriber({ token, sampleRate = 16000, onTranscript, onError })`
         - Build URL: `${WS_URL}?sample_rate=${sampleRate}&token=${token}&format_turns=true`
         - `const ws = new WebSocket(url)`
         - `ws.onmessage`: parse JSON, if `msg.type === 'Turn'`, call `onTranscript({ text: msg.transcript, isFinal: msg.turn_is_formatted === true })`
         - `ws.onerror`: call `onError?.('Voice connection failed')`
         - Return object:
           - `send(pcm16Buffer)` — if ws.readyState === WebSocket.OPEN, send buffer
           - `close()` — if ws.readyState === WebSocket.OPEN, send `JSON.stringify({ type: 'Terminate' })`, then ws.close()

    5. Append `getVoiceToken` server action to `lib/chat/actions.js` (VOICE-02, VOICE-04):
       - `export async function getVoiceToken()`
       - `await requireAuth()` (existing pattern, already defined in the file)
       - Dynamic import: `const { getConfig } = await import('../config.js')`
       - `const apiKey = getConfig('ASSEMBLYAI_API_KEY')`
       - If no apiKey, return `{ error: 'Voice input not configured' }`
       - `const res = await fetch('https://streaming.assemblyai.com/v3/token?expires_in_seconds=60', { headers: { Authorization: apiKey } })`
       - If !res.ok, return `{ error: 'Failed to get voice token' }`
       - `const { token } = await res.json()`
       - Return `{ token }`
       - Wrap in try/catch, return `{ error: 'Voice token error' }` on failure
       - NOTE: No audio data touches the server. Server only provides the temp token (VOICE-04).
