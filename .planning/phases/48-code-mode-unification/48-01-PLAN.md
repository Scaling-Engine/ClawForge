<objective>
Collapse three disconnected chat input toggles (Terminal, Shell, Code) into one unified Code toggle with Plan/Code sub-mode dropdown. Kill backtick-wrapping code mode. Guard Code toggle behind admin role.

Purpose: The current 3-toggle UX is confusing — "Code mode" wraps in backticks (useless) while "Terminal mode" routes to the actual SDK bridge. Users must know to pick Terminal, not Code, to get Claude Code CLI. This phase unifies them into a single intuitive toggle.

Output: Modified chat.jsx, chat-input.jsx, and chat-page.jsx with unified Code toggle routing to /stream/terminal.
</objective>
