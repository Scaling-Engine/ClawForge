<task type="auto">
  <name>Task 2: Create lib/tools/mcp-servers.js with loadMcpServers() and buildMcpConfig()</name>
  <files>lib/tools/mcp-servers.js</files>
  <action>
Create `lib/tools/mcp-servers.js` following the `loadAllowedRepos()` pattern from `lib/tools/repos.js`.

Export two functions:

**loadMcpServers():**
- Import `mcpServersFile` from `../paths.js` and `fs` from `fs`
- Read `mcpServersFile` synchronously, parse JSON
- Return `parsed.mcpServers` if it's an array, else return `[]`
- Catch all errors (file not found, parse error) and return `[]`
- Do NOT resolve template variables here — raw config only

**buildMcpConfig(servers):**
- Accept optional `servers` param, default to `loadMcpServers()` result
- Return `null` if servers array is empty
- For each server entry `{ name, command, args, env, allowedTools, hydrateTools }`:
  - Resolve template vars in `env` values only: replace `{{AGENT_LLM_*}}` patterns with `process.env[key] ?? ''` using regex `/\{\{(AGENT_LLM_[^}]+)\}\}/g`
  - Build Claude Code object map entry: `mcpServersObj[name] = { command, args: args || [], env: resolvedEnv }`
  - For each tool in `allowedTools` (default `[]`), push `mcp__${name}__${tool}` to toolFragments array
  - Collect `hydrateTools` entries separately for hydration step (MCP-08 in Plan 02)
- Return object:
  ```
  {
    configJson: JSON.stringify({ mcpServers: mcpServersObj }),
    allowedToolsFragment: toolFragments.join(','),
    hydrateSteps: allHydrateSteps  // array of { serverName, tool, args }
  }
  ```

Key constraints:
- Do NOT cache — always read fresh (env vars may change between dispatches)
- Do NOT strip `allowedTools` from the returned hydrateSteps (needed for Plan 02)
- The `configJson` output must use Claude Code's object map format `{"mcpServers": {"name": {...}}}`, NOT ClawForge's array format
- `hydrateTools` is optional on server entries; default to empty array
  </action>

'built' : 'null'); if (c) { const p = JSON.parse(c.configJson); if (!p.mcpServers) throw new Error('bad schema'); console.log('format OK, tools:', c.allowedToolsFragment); } })"</automated>
  </verify>
