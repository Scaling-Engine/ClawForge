Structure:
1. `'use client';` directive
2. Imports: `useState, useEffect, useCallback` from react; `useRouter` from next/navigation; `getAgentPickerData` from `../actions.js`
3. `formatRelativeTime(timestamp)` — copy verbatim from superadmin-dashboard.jsx
4. `StatusBadge({ status })` — copy verbatim from superadmin-dashboard.jsx
5. `AgentCard({ agent, onSelect })` component:
   - Renders a `<button>` wrapping the card (not a div) for accessibility, `type="button"`
   - `onClick`: calls `onSelect(agent.name)` then `router.push('/agent/' + agent.name + '/chat')`
   - Layout mirrors InstanceCard: `rounded-lg border bg-card p-4 space-y-3 w-full text-left hover:bg-accent/50 transition-colors cursor-pointer`
   - Offline agents: add `opacity-60` class when `agent.status !== 'online'` (still clickable per D-01)
   - Header row: agent name (font-semibold text-sm) + StatusBadge
   - Stats grid (3 cols): Active Jobs, Open PRs, Workspaces — use `text-2xl font-bold` for numbers
   - Footer: `Last job: {formatRelativeTime(agent.lastJobAt)}` in `text-xs text-muted-foreground`
6. `LoadingSkeleton()` — 2 skeleton cards, `h-32 rounded-lg bg-muted animate-pulse`
7. `AgentPickerPage()` main component:
   - State: `data` (array), `loading` (bool), `error` (string|null)
   - `loadData` with `useCallback` calling `getAgentPickerData()`
   - `useEffect` runs `loadData()` on mount (no polling interval — picker is not a live dashboard)
   - `onSelect(slug)`: sets `document.cookie = 'lastAgent=' + slug + '; max-age=2592000; path=/; SameSite=Lax'`
   - Empty state (data.length === 0, not loading): centered, muted icon (use a simple SVG or the existing UserIcon pattern), text "No agents assigned yet." on one line, "Contact your admin to get access." on second line in text-muted-foreground text-sm
   - Card grid: `grid gap-4 sm:grid-cols-2 lg:grid-cols-3`
   - Page wrapper: `<div className="max-w-4xl mx-auto p-6 space-y-6">`
   - Page heading: `<h1 className="text-2xl font-bold">Your Agents</h1>`

**cookie note:** Use `document.cookie = \`lastAgent=\${slug}; max-age=2592000; path=/; SameSite=Lax\`` — not httpOnly (must be readable by JS for the cookie-based persistence check).

**useRouter:** import from `'next/navigation'` and call inside the component function body (hooks rule).
