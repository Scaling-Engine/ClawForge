# Quick Task 260323-c16: Enable Claude Max Subscription Auth Summary

**One-liner:** Operator docs, admin badge (green/blue/orange), and entrypoint log line for switching job containers from API key to Claude Max OAuth token auth.

## What Was Built

### Task 1: Operator docs and env var reference (commit 276c4b0)

Added a complete "Claude Max Subscription Auth" section to `docs/OPERATOR_GUIDE.md` covering:
- Overview of `CLAUDE_CODE_OAUTH_TOKEN` as the alternative to pay-per-use `ANTHROPIC_API_KEY`
- Numbered setup steps: generate token with `claude setup-token`, add to `AGENT_LLM_SECRETS` JSON, remove API key, restart, verify
- Token lifecycle table (format, ~1 year lifetime, auto-refresh, re-auth procedure)
- Pitfalls: `--bare` flag skips OAuth, GitHub Actions dispatch path needs separate secret, monitor for expiry

Added `CLAUDE_CODE_OAUTH_TOKEN` row to the env var table in `docs/CONFIGURATION.md`.

### Task 2: Admin UI indicator and entrypoint log (commit 36daf76)

**`lib/chat/actions.js`** ‚Äî New `getJobAuthMethod()` server action that reads `AGENT_LLM_SECRETS` from the event handler's process environment and returns `{method, detail}`:
- `subscription` ‚Äî when `CLAUDE_CODE_OAUTH_TOKEN` is present
- `api-key` ‚Äî when `ANTHROPIC_API_KEY` is present (but no OAuth token)
- `none` ‚Äî when neither is present
- `unknown` ‚Äî when `AGENT_LLM_SECRETS` cannot be parsed

**`lib/chat/components/admin-general-page.jsx`** ‚Äî New "Job Container Auth" `ConfigSection` added as the first section on the general admin page. Shows a colored read-only badge:
- Green "Subscription" ‚Äî `CLAUDE_CODE_OAUTH_TOKEN` detected
- Blue "API Key" ‚Äî `ANTHROPIC_API_KEY` detected
- Orange "Not Configured" ‚Äî neither detected

**`templates/docker/job/entrypoint.sh`** ‚Äî Added auth method detection block after preflight echo lines. Logs `Auth method: subscription (CLAUDE_CODE_OAUTH_TOKEN)`, `Auth method: api-key (ANTHROPIC_API_KEY)`, or `Auth method: none detected`. Also adds `| Auth method | ${AUTH_METHOD} |` row to the `preflight.md` artifact table.
