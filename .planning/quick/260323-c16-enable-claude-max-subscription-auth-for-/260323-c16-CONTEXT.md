## Specific Ideas

From research:
1. `claude setup-token` generates a long-lived OAuth token (~1 year)
2. Add token to `AGENT_LLM_SECRETS` JSON as `CLAUDE_CODE_OAUTH_TOKEN`
3. Remove `ANTHROPIC_API_KEY` when switching to subscription
4. The `--bare` flag must NOT be used (it skips OAuth)
5. Operator can run `claude setup-token` in a ClawForge workspace terminal or on the VPS host
