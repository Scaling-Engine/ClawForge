**1. Add `fetchRepoFile()` to `lib/tools/github.js`:**

Add a new exported async function `fetchRepoFile(owner, repo, filePath, options)` that:
- Accepts `{ ref = 'main', maxChars = null }` options
- Uses direct `fetch()` to `https://api.github.com/repos/{owner}/{repo}/contents/{filePath}?ref={ref}`
- Sets headers: `Authorization: Bearer ${process.env.GH_TOKEN}`, `Accept: application/vnd.github.raw+json`, `X-GitHub-Api-Version: 2022-11-28`
- DO NOT use `githubApi()` -- it hardcodes the wrong Accept header (returns base64 instead of raw text)
- Returns `null` on 404 or 403 (not found / no access -- both expected for repos without .planning/)
- Returns `null` on any caught error (non-fatal -- log to console.error)
- On success: reads `res.text()`, truncates to `maxChars` with `\n\n[TRUNCATED]` suffix if over limit
- Add to the export block at the bottom
